# Mascot Style Notes

Public visuals use original lightweight illustration components instead of publishing Matix's source photos by default.

Reference assets:

- `source-photos/matix-pickle-pup-reference.jpg`: Matix with the black dog, used for Pickle Pup and founder warmth.
- `source-photos/matix-founder-pose-reference.jpg`: founder pose and confidence.
- `source-photos/matix-beach-cap-reference.jpg`: personality and face reference.
- `source-photos/matix-smile-reference.jpg`: smile/personality reference.

The source photos are not imported into the public app. If Mike later approves public photo use, add resized copies to `public/assets/` and document where they appear.

Generated public artwork:

- `matix-pickle-crew-hero.jpg`: wide hero illustration with Matix, Pickle Pup, pickle mascots, jars, and cucumber garden.
- `matix-founder-card.jpg`: portrait card of Matix holding a jar.
- `pickle-business-plan-board.jpg`: blank-panel business plan board retained as reference only. The live business-plan section uses real HTML text so it never appears empty on mobile.

The generated images intentionally avoid readable text so site copy remains editable, accessible, and accurate in HTML.
