# Setup Guide

## Requirements

- Node.js 22 or newer.
- npm.

## Install

```bash
npm install
```

## Run Locally

```bash
npm run dev
```

Open the local URL printed by Next.js.

## Verify

```bash
npm test
npm run check
npm run build
```

`npm test` checks business logic. `npm run check` checks TypeScript. `npm run build` verifies the production app can compile.
