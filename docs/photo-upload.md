# Photo Upload Guide

Source photos are stored in:

```text
source-photos/
```

These are reference assets, not public site assets.

## Public Photo Rule

Do not publish real photos of Matix by default. Use illustrated/cartoon versions first. If Mike later approves public photo use:

1. Resize and optimize the selected image.
2. Place the public copy in `public/assets/`.
3. Avoid filenames that expose private details.
4. Document exactly where the photo appears.
5. Re-run mobile checks to confirm cropping is respectful and intentional.
